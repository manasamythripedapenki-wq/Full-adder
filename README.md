Description
This project implements a 1-bit Full Adder using Verilog HDL. A Full Adder adds three binary inputs (A, B, Cin) and produces two outputs (Sum, Cout)
## Features
. Designed using Verilog HDL
. Generates Sum and Carry outputs
. Verified using a testbench and waveform simulation
. Suitable for digital and VLSI design learning
Tools Used
. Verilog HDL
. EDA Playground
. GTKWave
Outputs
. Sum = A ⊕ B ⊕ Cin
. Cout = (A & B) + (B & Cin) + (A & Cin)
